First Quarter of 2020Release Date: March 27, 2020This topic provides a list of product changes released for this module on this date.
Table 1. Resolved Issues
The following issues, which existed in one or more previous versions, have been resolved.
DescriptionTracking IDPreviously, users with certain permission settings were unable to access the application navigation menu and start page. This issue has been resolved. Now, you can assign the "APM Basic User Permission" permission set to ensure that users are able to access the menu and start page regardless of other assigned permissions. The "APM Basic User Permission" permission set should be applied to all users for all resources in the tenant.US378220
