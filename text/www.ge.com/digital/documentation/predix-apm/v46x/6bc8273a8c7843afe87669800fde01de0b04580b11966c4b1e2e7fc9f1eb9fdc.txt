Second Quarter of 2023Release Date: June 29, 2023This topic provides a list of product changes released for this module on this date.
Table 1. Enhancements and New Features
The following enhancements and new features have been added.
DescriptionTracking IDTo enhance usability and productivity of Analysis charts, an error message appears if there is a Unit of Measure (UoM) conversion failure. This helps in preventing you from continuing your analysis based on incorrect assumptions and incomplete data. The following enhancements have been added:Error message appears in the chart and the deck notification bar.You can either reload an individual chart or reload all failed charts from the deck error banner. If UoM conversion fails for the second time, you are advised to refresh the page.F71558
