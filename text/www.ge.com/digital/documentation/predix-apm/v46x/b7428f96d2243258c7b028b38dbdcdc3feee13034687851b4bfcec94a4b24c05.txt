Third Quarter of 2018SearchThis topic provides a list of product changes released for this module on the dates listed below.Release Date: September 21, 2018
Table 1. Enhancements and New FeaturesSearch ConfigurationNoteTracking ID(s)The Search feature has been enhanced to allow users to configure the search index to provide more precise results based on customer data. To facilitate this enhancement, in the Search Configuration page, a new button, Settings, has been added. After selecting this button, in the Edit Settings window, an Administrator can now adjust the values in the Minimum Index Size and the Maximum Index Size drop-down list boxes. These fields determine how the Search indices are created. US280404
