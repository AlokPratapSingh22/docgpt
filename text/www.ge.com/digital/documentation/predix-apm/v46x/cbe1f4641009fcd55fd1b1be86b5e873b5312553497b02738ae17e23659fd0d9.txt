CardsExport a CardAbout This Task
You can export a card into multiple formats to share it with others using any means of communication outside of GE Digital APM. You can also attach it as an evidence to an Alert or Case. The export feature allows you to perform the following actions:Export as a PNG imageExport as a PDF documentCopy to clipboard
ProcedureNavigate to the desired card. For SmartSignal cards, ensure that all the charts are loaded before trying to export the card.In the Analysis card, select  to access the options, and then select Export All Charts On This Card. For a SmartSignal card, in the card which you want to export, select .The export options appear.Select one of the options listed in the following table:
OptionResultAs an ImageA message appears indicating that the images are being created and it may take time depending on the number of charts in the card. After the completion of the process, a PNG file is downloaded to the browser that includes images of all the charts in the card. The tag registers on the images of each chart displays the tag names and the asset names.Note: For Smart Signal charts, all the tag names are grouped at the top of the image as they are common across all the charts. This provides a compressed view of all the charts. The tag registers on the images of each chart displays only the tag names.As a PDFA message appears indicating that the PDF file is being created and it may take time depending on the number of charts in the card. After the completion of the process, a PDF file is downloaded to the browser that includes images of all the charts in the card. The tag registers on the images of each chart displays the tag names and the asset names.Note: For Smart Signal charts, all the tag names are grouped at the top of the image as they are common across all the charts. This provides a compressed view of all the charts. The tag registers on the images of each chart displays only the tag names.To ClipboardA message appears indicating that the images are being created and it may take time depending on the number of charts in the card. After the completion of the process, all the chart images are copied to the clipboard. You can then paste the contents of the clipboard to any application. The tag registers on the images of each chart displays the tag names and the asset names.Note: For Smart Signal charts, all the tag names are grouped at the top of the image as they are common across all the charts. This provides a compressed view of all the charts. The tag registers on the images of each chart displays only the tag names.

Note: If you try to embed a large set of charts that have been saved as an image or copied to a clipboard in the body on an email message, Outlook compresses any images beyond a certain resolution that leads to a distorted image in the email for the recipient. To avoid this issue, it is recommended to attach the images to the email if the number of charts is more than four.
