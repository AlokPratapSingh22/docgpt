Fourth Quarter of 2020Release Notes: December 11, 2020This topic provides a list of product changes released for this module on this date.
Table 1. Enhancements and New Features
The following enhancements and new features have been added.
DescriptionTracking IDThe chart controls are enhanced for usabilty. To facilitate this enhancement, the following changes have been implemented:Selection of common chart controls such as zoom, cross-hair, and so on is applied across all the charts in a card and also across all the charts in any other card in the view. Any new charts added to the cards also retain the common chart controls selection made in one card. Deleting the chart or the card or moving to another card does not change the common chart controls selection. The sub configuration options of the common chart controls do not open for all the charts on which the selection is propagated.The common chart control selections are reset when you select another chart control outside the common controls and refresh the chart.When navigating to Analysis from an Alert, the selection of common chart controls is retained on the charts. The selection of common chart controls is retained on the Event Data Analysis charts.F56453The Search Browser pane for tag search results is now enhanced for usability and efficiency. To facilitate this enhancement, the following changes have been implemented:The Search button now appears below the Search box and the area of the Search box has been increased to allow more search criteria entries. The Search button is enabled only when any search criteria are added to the Search box. The Search button is disabled after the search is triggered until a new criterion is added or any existing criteria is modified.Hovering over the search criteria chip displays an information tooltip prompting you to double click the tag to modify the search criterion or click the Cancel () icon to clear the criterion.You can now search for up to 100 tags in a single search that are copied from a .CSV file or a notepad to the Search box. To facilitate this enhancement, pagination is now introduced so that you can view a maximum of 40 search results in each page.If all the search results appear on one page, the Select All check box now appears on the Search Browser. You can select all the tags from the search results to add to a chart, by selecting this check box.If the search results appear on multiple pages, the Select Visible Tags check box now appears on the Search Browser. You can select all the visible tags on the page to add to a chart by selecting this check box.You can now select multiple tags from the tag search results to add to a chart, by using the Shift key or the Control key on the keyboard. You can select a maximum of 40 tags.The search results header now displays the number of search results that are selected.The Search box scrollbar now works only when you enter more than 10 search criteria in the box.The search help icon and the help text have been updated.F56452To enhance usability in charts, the Delete Chart () icon is now added to the chart controls.
