Filter the Context BrowserYou can filter within the Context Browser selections by asset name.About This Task
The filter capability for the Context Browser allows you to select an asset classification via the business functional hierarchy. 
During selection of an asset context, you can filter the Context Browser list by asset name.
The Context Browser contains columns of assets that are organized into business functional object groups by Enterprise, Site, Segment, and asset context groups by Asset. Each column contains asset names that can be selected and opened to select your asset. You can filter inside a column to help you find the asset context you want to use.
ProcedureIn the module navigation menu, navigate to Assets, Analysis, or Dashboards.Select the Context Browser.In the Select Context window, select .The Asset Classification drop-down list appears.Select an option from the Asset Classification drop-down list, and then select Apply.All matching instances of the asset name appear in the search results.In the Select Context window, select an option from the list, and select Select.The asset context appears at the top.
