The following issues, which existed in one or more previous versions, have been resolved.
DescriptionTracking IDPreviously, if your  system used an Oracle schema, when you attempted to sort the execution history on the Summary column, an error occurred and the sorted list was not displayed. It is not possible to fully address this issue due to performance limitations. Now, if your  system uses an Oracle schema, when you attempt to sort on the Summary column, a message is displayed requesting you to sort on a different column.DE191242
